Large HexaRoll No.1

This page features Oskar's original HexaRoll puzzle. If you solve this one try another.

Sorry, this is a JAVA applet. To run it you must install a suitable browser extension such as the CheerPj applet runner for Edge or Chrome. Please see the FAQ.

Aim

Spin the board until the ball comes to rest in the yellow target area.

Controls

Click on Restart to reset.

Movement

Use the spin buttons to rotate the board. The board will not spin while the ball is in motion.
