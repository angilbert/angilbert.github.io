The original HTML maze

This is a maze constructed entirely using HTML (tables). It was introduced Feb 1998, dropped March 2000 and reinstated May 2001 (to accompany Oskar's Four Bit mazes). Like the Four Bit mazes this maze has just 16 states. If you get stuck in a dead-end click here to restart.
