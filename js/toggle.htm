The toggle button maze

Use the arrow buttons to guide the central square around the 5x5 grid and collect the goals in all four corners. But beware you can only use each button once until you can find a way to toggle it back on again and each button either toggles or disables another pair of buttons. Can you find a safe route without disabling all the buttons?

 

Aim

Guide the central square around the 5x5 grid and collect all four corner goals.

Controls

Use the big coloured buttons to move the square.
You can only use the enabled buttons.
Reload the page to restart.

Hint

Don't disable all the toggle buttons at the same time!
