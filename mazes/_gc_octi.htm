Octiamond Maze
Inspired by Ed Pegg's Octiamonds puzzle set (www.mathpuzzle.com). Start at the top in the direction of the solid blue arrow. Return to the top in the reverse direction. At each red node your angle of departure must be different to the last. That is, never use sharp, soft or straight twice in a row (U-turns are never permitted).
